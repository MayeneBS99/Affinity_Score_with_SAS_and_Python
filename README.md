
# Customer churn prediction using Logistic Regression with SAS and Python : Overview 

![Intro](images/scoring_main_picture.webp)

L’objectif de la direction Marketing de l’entreprise “Envies d’Ailleurs” sur l’année à venir est 
de Déterminer les potentiels de croissance de l’entreprise.  
Pour ce faire, il faudra établir des analyses de cibles et proposer des actions commerciales 
de développement. 
Notre objectif dans cette étude sera dans un premier temps de déterminer les critères de 
ciblages des prochaines offres de séjour pour célibataire et ensuite de calculer le nombre de 
clients à cibler, pour envoi des messages par internet. 

Dataset de depart 6387 lignes et 326 variables

**Tools and Technologies  :** 

>- **SAS** (PROC UNIVARIATE, PROC FREQ, PROC LOGISTIC)
>- **Python** and **Streamlit** for code industrialization and deployement
>- **Git** for version control;
